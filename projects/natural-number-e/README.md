# 🌟 自然之数 - 探索欧拉常数 e

> 从银行利息到宇宙规律的数学冒险之旅

一个面向高一学生的交互式数学学习项目,通过图形、文字、声音、计算机动画等多种形式,深入浅出地介绍自然常数 e 的发现史、数学地位、有趣故事和特殊性质。

## ✨ 项目特色

### 🎮 交互式学习
- **参数实时调节**: 拖动滑块,观察复利计算中 e 的诞生
- **可视化图表**: ECharts + Three.js 呈现数学之美
- **动画演示**: GSAP 动画展示数学概念的变化过程

### 🎨 手绘风格设计
- **活泼生动的界面**: Comic Sans MS + Ma Shan Zheng 字体
- **拟人化小猫吉祥物**: 7种情绪状态,陪伴你的学习旅程
- **纸张质感背景**: 温馨的手绘笔记本风格

### 🤖 AI 学习助手
- **23+ 主流 AI 模型支持**:
  - 国际: OpenAI GPT-4, Claude 3, Gemini Pro, Llama 3, Mistral, Cohere
  - 国产: 智谱 GLM, 百度文心, 阿里通义, 讯飞星火, Moonshot, Baichuan, MiniMax
  - 新增: **DeepSeek** (强逻辑推理,顶级数学能力)
- **自定义模型**: 支持配置任意兼容 OpenAI API 的模型
- **本地存储 API Key**: 数据仅保存在浏览器,安全可靠
- **上下文智能对话**: AI 会根据当前学习内容提供针对性解答

### 🔗 概念关联系统
- **微积分**: 导数、积分的直观理解
- **欧拉公式**: e^(iπ) + 1 = 0 的数学美
- **泰勒级数**: e 的级数展开
- **正态分布**: e 在概率论中的应用
- **泊松分布**: 稀有事件的数学模型

### 🎵 沉浸式体验
- **背景音乐**: 配合学习节奏的轻音乐
- **交互音效**: 点击、滑动、发现的即时反馈
- **成就系统**: 完成学习任务,收集徽章

## 🚀 快速开始

### 环境要求
- Node.js >= 18.0.0
- npm >= 9.0.0

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本
```bash
npm run build
```

### 预览生产构建
```bash
npm run preview
```

## 📝 开发规范

### 命名约定
- **组件**: PascalCase (e.g., `MascotCat.vue`)
- **文件**: kebab-case (e.g., `model-registry.ts`)
- **变量**: camelCase (e.g., `showAIPanel`)
- **常量**: UPPER_SNAKE_CASE (e.g., `MODEL_REGISTRY`)

- `refactor`: 代码重构
